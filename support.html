❓ Frequently Asked Questions

What languages does the voice recognition support?

+

The AI primarily supports English, but it can understand financial terms and counterparty names in multiple languages. For best results, use English for the primary trade description and industry-standard terminology (ISIN, DVP, FOP, etc.).

Is my voice data stored anywhere?

+

No. Your audio recordings are sent to Anthropic's AI for real-time processing only. They are not stored, retained, or used for training. Once the AI extracts the trade information, the audio is discarded. See our Privacy Policy for full details.

Can I use the app without voice recording?

+

Absolutely. You can manually fill all 20+ fields using the form view. The voice recording is optional — it simply speeds up data entry. All dropdown fields and text inputs work independently of the AI feature.

What file format does the Excel export use?

+

The app exports files in .xls format compatible with Microsoft Excel, Apple Numbers, Google Sheets, and LibreOffice Calc. The export includes color-coded group headers, dropdown data validations, conditional formatting for status and priority, and a separate Dashboard sheet with KPI summaries.

How many trades can I manage at once?

+

There's no hard limit. You can add as many trades as needed using the "Add Trade" button. Each trade maintains its own set of 20+ fields. Switch between trades using tabs in form view, or see all trades simultaneously in table view.

Does the app work offline?

+

Manual data entry and Excel export work fully offline. Voice recording requires an internet connection since the AI analysis is performed in the cloud. If you record while offline, the app will use demo data as a fallback.

What trade fields are supported?

+

PocketBA supports 20+ fields organized in 4 groups: Trade Details (Trade ID, Counterparty, Trade Date, Settlement Date, Asset Class), Instrument (Type, ISIN/CUSIP, Currency, Notional, Quantity, Price), Settlement (Buy/Sell, Method, CSD, SSI Details, Broker, Custodian), and Status (Status, Priority, Notes). Fields with standardized values have dropdowns.

Can I customize the Excel export template?

+

The current version uses a professionally designed template with industry-standard formatting. Custom templates are planned for a future update. The exported file is fully editable in Excel, so you can adjust formatting after export.
